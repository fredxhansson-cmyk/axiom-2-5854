# Mindfulness Kursportal

Denna applikation är en onlineportal för att tillhandahålla kurser i mindfulness och stresshantering.

## Kom igång

1. Klona repot.
2. Installera beroenden med `npm install`.
3. Konfigurera dina miljövariabler i `.env` (se `.env.example`).
4. Kör appen lokalt med `npm run dev`.

## Funktioner

- Anpassad design med Lora font och blå accentfärg.
- Användarvänligt gränssnitt med Next.js.
- Integrerat med Stripe för betalningar.
- SEO-optimerad för att öka synlighet online.
